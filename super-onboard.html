Onboard a New Landlord in Minutes

Follow these steps to invite a landlord, get them set up on LandlordHQ, and make sure their first tenant is live before you're done.

Step-by-step
How to onboard a new landlord
1
Open the Super Admin Hub
Navigate to /super on your LandlordHQ instance. This is your command center for managing all landlord accounts across the platform.
/super
The Super Admin Hub is only accessible to the system owner. Keep this URL private.
2
Generate an Invite Link
In the Super Hub, click "Generate Invite". A unique one-time signup link will be created for the new landlord. This link is tied to a single account creation — it expires after use.
Each invite link is single-use. Generate a fresh one for each new landlord.
3
Send the Link to the Landlord
Copy the invite link and share it with the landlord via Telegram, email, or any secure channel. Let them know they'll need to choose a username to complete signup.
Tell them to open the link on a browser — not inside Telegram's in-app browser — for the smoothest experience.
4
Landlord Opens the Signup Page
The invite link takes the landlord directly to signup.html — the registration page. They fill in three fields to create their account:
Invite Code — auto-filled & locked from the link (e.g. INV-XXXXXX)
Full Name or Company Name — e.g. "John Doe Properties"
Desired Username — this is their login handle
After submitting, they'll receive a confirmation with a /claim command to send to the Telegram bot, linking their account to Telegram.
Once the bot confirms, their account is live. You'll see them appear in the Super Hub under registered landlords.
5
Guide Them to Add a Property
Walk the landlord through adding their first property — name, address, city, state, number of units, and status. Refer them to the Landlord Onboarding Guide for a full walkthrough.
Properties are the foundation. Tenants must be linked to a property before they can receive rent reminders.
6
Landlord Is Live
Once their first tenant is added and linked via Telegram, rent reminders and payment tracking are fully automated. Your job is done — the system handles the rest.
You can monitor all active landlord accounts from the Super Hub at any time.

Telegram Commands Reference

Full command list for @landlord_hq_bot — all roles

Super Admin note: Super admin management — generating invite codes, viewing the audit log, and overseeing all landlord accounts — is done entirely through the web dashboard at /super. Bot commands below are shared with landlords and tenants.
Landlord Commands
/claim <username> Claim a landlord account on Telegram after signing up on the web
/verifyclaim <username> <code> Verify the claim with a 6-digit code to complete account activation
/addtenant <unit> <name> [dueday] Add a new tenant — generates a link code to share with them (default due day: 1)
/removetenant <unit> Remove a tenant unit and all their associated data
/tenantlist View all tenant units — shows name, due day, and Telegram link status
/broadcast <message> Send an announcement to all linked tenants at once
/myid Get your Telegram chat ID — needed for the Fixer Chat ID setting in the dashboard
Tenant Commands
/start Show the welcome message
/help Show all available bot commands
/link <code> Link a rental unit using the code provided by the landlord
/rules View the house rules for the property
/report <issue> Submit a maintenance or concern ticket — can attach a photo or video as evidence
/payment Submit a payment receipt — send a photo or video with /payment in the caption
/clearance View the move-out clearance process
All commands are used directly in the @landlord_hq_bot Telegram chat.
